# рџ“Љ Key Insights

- Movies account for nearly 70% of Netflix content.
- TV Shows are growing rapidly in recent years.
- United States has the highest number of titles.
- Dramas and International Movies dominate the catalog.
- TV-MA is the most common content rating.
- July and December have the highest content additions.
- Average movie duration is around 100 minutes.
